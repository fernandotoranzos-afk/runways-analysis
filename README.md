# runways-analysis
Data analysis project using Python and Power BI
 Análisis de Infraestructura Aeroportuaria
Proyecto de análisis de datos enfocado en la exploración y visualización de características de pistas de aeropuertos a nivel global.
________________________________________
 Objetivo
Analizar la estructura de pistas aeroportuarias en función de:
•	Tamaño (clasificación propia) 
•	Longitud 
•	Área 
•	Iluminación 
________________________________________
Tecnologías utilizadas
•	Python (Pandas) 
•	Jupyter Notebook 
•	Power BI 
________________________________________
 Proceso
1. Limpieza de datos
•	Normalización de columnas 
•	Tratamiento de tipos de datos 
•	Eliminación de inconsistencias 
2. Transformación
•	Escalado (length_norm) 
•	Codificación categórica (surface_code) 
•	Feature engineering (area, size_category) 
3. Visualización
Dashboard en Power BI dividido en:

 Página 1 — Overview
•	Distribución de pistas por tamaño 
•	Longitud promedio por categoría 
•	Composición por iluminación 
•	Proporción de iluminación 
 Página 2 — Análisis profundo
•	Relación entre longitud y área 
•	Impacto de iluminación en longitud 
________________________________________
 Insights clave
•	La mayoría de las pistas pertenecen a categorías pequeñas y medianas. 
•	Las pistas grandes presentan mayor longitud promedio. 
•	Existe una relación directa entre longitud y área. 
•	Las pistas iluminadas tienden a ser más largas, lo que indica mayor infraestructura.
